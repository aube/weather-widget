# weather-widget

Vue2 web-component weather-widget.

App state implemented via Vue.observable.

Data from api.openweathermap.org.


## Example

[demo.html](https://raw.githubusercontent.com/aube/weather-widget/master/dist/demo.html)

```
npm install
```
